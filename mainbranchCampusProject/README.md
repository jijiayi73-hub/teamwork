# CampusProject

Project workspace for the CampusProject application.

## Structure

- `frontend/`: client application
- `backend/`: server application
- `database/`: database schema, seed data, and local SQLite database
- `docs/`: technical, architecture, API, testing, and defense documentation
- `prompts/`: prompt assets and AI workflow notes
- `vibe/`: design direction and product tone references
- `scripts/`: development and maintenance scripts
- `demo/`: demo assets and walkthrough materials
