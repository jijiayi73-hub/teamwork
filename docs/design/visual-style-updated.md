# 视觉风格文档

项目名称：心灵情绪日记 / Mindful Memory Diary / Inner Garden

## 1. 总体视觉定位

心灵情绪日记的视觉方向是“治愈系梦境花园”。界面应该给用户安静、柔软、被接住的感受，让记录情绪这件事不显得沉重，也不显得医学化。

视觉关键词：

- 治愈系水彩
- 淡油画
- 梦境花园插画
- Dreamcore nature
- Ethereal surrealism
- 低饱和
- 柔和光感
- 半透明水彩
- 植物隐喻
- 手账感排版

## 2. 风格方向

### 2.1 治愈系水彩

用于整体界面背景、记忆卡片封面、空状态插画和轻量装饰。水彩质感应半透明、边缘柔和、层次自然，避免高对比和过度锐利。

适用场景：

- 首页背景。
- Diary Result 封面生成。
- Memory Garden 卡片封面。
- 情绪底色。

### 2.2 淡油画

淡油画用于增强记忆卡片的收藏感。画面可以有轻微笔触和柔光，但不应过重或过暗。

适用场景：

- AI 生成记忆封面。
- Memory Detail 头图。
- 特殊情绪卡片，例如释然、平静、疲惫。

### 2.3 梦境花园插画

梦境花园是产品差异化视觉核心。每一张记忆卡片都可以像花园中的一片区域、一朵花、一条小路或一束光，暗示当天情绪被温柔保存。

常用元素：

- 花朵
- 枝叶
- 草地
- 小径
- 雾气
- 光斑
- 玻璃温室
- 水面倒影
- 小纸片和手账贴纸

### 2.4 Dreamcore nature

Dreamcore nature 用于营造轻微梦境感和非现实空间感，但要克制。画面可以像真实自然和梦境之间的空间，不做恐怖、压迫或迷幻过强的表达。

可用表现：

- 柔和雾气。
- 不完全真实比例的花园。
- 漂浮的纸张、光点或植物。
- 远处朦胧的门、窗、温室或小路。

### 2.5 Ethereal surrealism

Ethereal surrealism 用于 Memory Detail 和封面生成，让回忆有一种轻盈、超现实但安全的质感。

表现原则：

- 空灵、柔和、低压。
- 超现实元素只作为隐喻，不喧宾夺主。
- 不使用黑暗、恐怖、破碎或强刺激画面。

## 3. 色彩规范

### 3.1 总体色彩

整体采用低饱和、柔和光感的色彩。避免大面积高纯度颜色，避免强烈红黑对比，避免医疗感蓝白配色。

推荐主色：

- 雾粉色：用于温柔、委屈、安静氛围。
- 鼠尾草绿：用于花园、恢复、平静氛围。
- 浅天蓝：用于呼吸感、空间感。
- 奶油白：用于背景和纸张感。
- 淡薰衣草紫：用于梦境感，但不大面积使用。
- 柔雾灰：用于次级文字和边框。

### 3.2 情绪底色建议

- 开心：柔和杏黄、浅桃色。
- 平静：鼠尾草绿、浅蓝灰。
- 疲惫：雾灰紫、低饱和蓝。
- 焦虑：浅灰蓝、淡薄荷绿。
- 难过：雾蓝、浅灰紫。
- 委屈：雾粉、淡玫瑰色。
- 愤怒：低饱和珊瑚红，不使用强烈纯红。
- 释然：浅绿、奶油白、柔光金。
- 空白：米白、淡灰、透明水彩纹理。

## 4. 图像生成风格

默认图片风格：

> 治愈系水彩 / 淡油画 / 梦境花园插画，低饱和色彩，柔和光感，半透明水彩边缘，植物隐喻，手账感排版，安静、温柔、可回忆。

使用原则：

4. 所有情绪图片仍需遵守“治愈系梦境花园”的总体视觉方向，避免恐怖、压迫、强刺激和医疗诊断感。

### 4.2 情绪收藏夹到生成风格的映射

- 怀念：适合生成带有回忆感、柔雾光、旧照片温度、花园小路、纸张碎片、远处窗户或温室的画面。
- 平静：适合生成浅绿色、浅蓝灰、清晨柔光、水面倒影、安静植物和透明温室的画面。
- 焦虑：适合生成浅灰蓝、淡薄荷绿、轻雾、微微缠绕的植物或窄小路径，但画面必须保持安全、柔和、可被安抚。
- 喜悦：适合生成柔和杏黄、浅桃色、阳光花朵、轻盈光点、开放花园和温暖手账感画面。
- 封面：用于主界面视觉，应像进入一座梦境花园的入口，强调第一眼的温柔、治愈、记忆感和产品识别度。

生成流程建议：

1. AI 先生成背景图，风格为治愈系水彩、淡油画或梦境花园插画。
2. 前端 Canvas 或页面层叠加日记标题、日期、情绪标签和简短摘要。
3. 用户保存后，最终封面作为 Memory Garden 的记忆卡片图。

尺寸建议：

- 电脑端：1500 x 1200 px，比例 5:4，适合电脑和平板查看。
- 手机端：500 x 700 px，比例 5:7，适合移动端快速加载和浏览。

## 5. 页面视觉要求

### 5.1 Home

首页要像一座安静花园的入口。背景可以使用柔和水彩或梦境自然插画，主标题清楚可读，按钮不应过于鲜艳。主界面使用“封面”Pinterest 收藏夹作为照片和插画生成参考，绘制方向应是梦境花园入口、柔光植物、温室、小路、纸张感和安静空间感。

视觉重点：

- 第一屏直接体现产品名称和花园意象。
- 主按钮清晰，但风格柔和。
- 首页背景参考“封面”收藏夹重新绘制，不直接使用 Pinterest 原图。
- 不使用医疗、测评、诊断类视觉元素。

### 5.2 AI Companion Chat

聊天页要让用户感觉正在被认真倾听。界面应简洁，消息气泡圆角适中，留白充足。

视觉重点：

- 用户气泡和 AI 气泡颜色区分，但都保持低饱和。
- 输入区固定在底部，清晰易用。
- 上传图片预览像手账贴纸，不做强边框。

### 5.3 Diary Result / Mood Check-in

日记生成页要像一张正在完成的手账页。结构清楚，允许用户编辑，不应让 AI 结果显得不可修改。

视觉重点：

- 日记正文区域有纸张感。
- 情绪标签像柔和贴纸。
- 封面预览要突出画面和保存感。

### 5.4 Memory Garden

Memory Garden 是视觉中心。它应像回忆相册和情绪花园结合，而不是普通数据列表。

视觉重点：

- 卡片以封面图为主，文字信息简短。
- 卡片之间留有呼吸感。
- 可以使用花园路径、分区、月份等隐喻组织内容。

### 5.5 Memory Detail / Past Self Chat

详情页应有“翻开某一天”的感觉。左侧日记像保存下来的手账，右侧对话像给过去自己的留言。

视觉重点：

- 记忆封面和日记内容占据主要视觉。
- 对话区不抢占日记主体。
- 推荐问题用轻量按钮呈现。

## 6. 排版规范

- 标题清楚、温柔，不使用强冲击字体。
- 正文行距适中，适合长段日记阅读。
- 手账感排版可以使用小标签、小日期、小分隔线，但不要过度装饰。
- 按钮文字简短，例如“开始记录”“保存到 Memory Garden”“返回继续聊”。
- 页面文案避免说教，避免评价用户。

## 7. 视觉禁忌

- 不使用医院、诊断、测评量表等强医疗视觉。
- 不使用恐怖、压迫、黑暗、破碎感强的梦核画面。
- 不使用高饱和大红大紫作为主视觉。
- 不用复杂图表作为 MVP 主要界面。
- 不把用户情绪视觉化为病症或风险等级。

## 8. 示例封面提示词

平静：

> 参考“平静”收藏夹风格，低饱和水彩梦境花园，清晨柔光，浅绿色植物，小路通向透明温室，画面安静，有手账纸张质感，淡油画笔触，温柔、空灵、治愈。

怀念：

> 参考“怀念”收藏夹风格，柔雾水彩梦境花园，旧照片般的奶油白和浅金色光感，远处有模糊窗户、小路和飘落纸片，植物轻轻包围，像被保存下来的回忆，温柔、安静、可回望。

焦虑：

> 参考“焦虑”收藏夹风格，低饱和浅灰蓝和淡薄荷绿水彩花园，轻雾中的细小植物和微微弯曲的小路，画面表达不安但不压迫，有柔光从远处透进来，安全、被安抚、治愈。

喜悦：

> 参考“喜悦”收藏夹风格，柔和杏黄色和浅桃色梦境花园，阳光洒在小花和草地上，轻盈光点漂浮，手账纸张质感，低饱和、温暖、明亮但不刺眼。

疲惫：

> 柔和雾蓝色水彩插画，夜晚安静花园，一盏小灯照着长椅，植物轻轻围绕，低饱和色彩，半透明水彩边缘，淡淡油画质感，表达累但被接住。

释然：

> 梦境自然花园，浅金色柔光穿过树叶，白色小花和透明水面，低饱和绿色与奶油白，空灵超现实风格，温柔、轻盈、像一页手账封面。

主界面封面：

> 参考“封面”收藏夹风格，梦境花园入口，柔和水彩和淡油画质感，透明温室、植物小路、奶油白背景、浅绿色和淡粉色光感，第一眼安静、温柔、有记忆花园的入口感，适合作为 Inner Garden 首页主视觉。
